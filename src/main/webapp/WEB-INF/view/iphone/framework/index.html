云投资组合系统

用户可以通过本系统获检索投资组合，查看组合详情，查看关于本组合的微投研报告；用户可以选中、收藏自己认为好的组合，以便长期关注该组合的运行。用户可以查看该组合的历史曲线。 策略艾真体服务基于传统分析、人工智能分析，采用机器学习、基于神经网络的深度学习等方式生产投资组合，借助hbase、hive、hdfs等大数据技术体系进行数据存储与服务。

功能列表
  • • 投资组合 根据投资范围内的标的资产（比如所有a股股票），并通过人工智能的方式生成投资组合。
  • • 业绩报告 每日净值曲线、单位净值、资产总值、浮动盈亏等；
  • • 投研报告 技术参数、统计参数和频谱特征 比如 移动平均线（SMA）、平滑异同移动平均线（MACD）、标准差度量（std）、能量互信息等；
  • • 收藏 收藏个人看好的投资组合以便长期跟踪；
应用场景
  • 金融机构、券商、基金公司
  • 根据最新行情同步生产投资组合
  • 每日估值、投研分析报告

IA交易系统

从投资组合被确认投资后，我们系统即开始对该投资组合生命周期进行管理。系统主要以API的形式对外提供服务。系统可以接收外部发送的建仓、调仓、平仓等请求，接到该请求后会发往券商交易系统。此处所说的外部系统，可以是我公司其他关联产品系统，也可以是客户IT体系内其他系统；根据投资组合自身的投资周期属性，到期时自动发起平仓操作。

功能列表
  • • 接收投资组合 外部系统通过接口方式将投资组合按照特定报文格式传输到本系统。
  • • 接收交易指令 外部系统通过接口发送建仓、调仓、平仓等交易指令；
  • • 管理投资组合 管理投资组合生命周期、自主处理到期的投资组合；
应用场景
  • 金融机构、券商、基金公司
  • 受理交易指令并管理投资组合生命周期

ITS估值系统

  • • 接口服务 估值API接口、系统集群、交易数据接口；
应用场景

资金管理规划系统

主要受理用户较大资金、较长周期的理财规划。用户无需太多操作，也不用在海量投资组合中进行甄别、选择，只要输入投资金额、投资期限、期望收益、承担风险等信息，我们系统就会自动为用户选择投资策略，分配资金，调整仓位。每一组投资策略都有固定期限，当一批投资策略到期后，系统会根据当前投资的收益状态，选择第二批、第三批等投资策略。

功能列表
  • • 投资请求 用户设置投资参数（金额、期限、预期收益率、风险阈值）。
  • • 资金规划 长期（一个月以上）为用户规划投资策略、管理仓位；
  • • 风险监控 投资组合期限监控、收益监控、资金仓位监控；
  • • 报表 收益报表、交易数据报、现金流报表；
应用场景
  • 金融机构、券商、基金公司
  • 长期投资规划

交易风险管理系统

本系统主要实现对投资组合的风险管理。该系统主要以API的形式对外提供服务。 外部系统（可以是我单位的其他系统或客户方的系统）通过接口的形式将投资组合、当前持仓等信息输入到本系统，本系统根据实时行情，采用基于机器学习的算法，评判该投资组合的风险，对投资组合整体或部分股票给出平仓或继续持有的建议。客户也可以设定硬性指标（比如止盈线、止损线等），约束本系统的行动。

功能列表

  • • 行情监控 本系统对行情进行实时监控，控制投资组合仓位等；
  • • 监控数据 收益监控、风险监控、投资组合期限监控；
应用场景
  • 金融机构、券商、基金公司
  • 实现对投资组合的风险管理

智能投顾业务合规报告系统

人工智能投顾策略，如果运用不当，易引起股市的趋同效应。我们系统即对智能投顾交易进行评测，给出合规、有风险倾向、有明显风险等报告数据。系统通过数据采集接口，获取连续多日的智能投资策略的建仓、调仓、平仓等信息；并通过第三方行情系统，获取行情交易数据。系统通过分析当日、5日、10日等数据，给出自身机构的交易趋同度、自身机构交易与整个行情交易的资金占比等信息，判断是否对股票市场造成冲击，并别给合规、有风险倾向、有明显风险等报告数据。

功能列表
  • • 获取交易数据 通过数据接口获取智能投顾交易数据；
  • • 分析交易数据 分析1日或多日交易数据趋同度，及对市场影响等；
  • • 审核意见 以图形界面或接口的形式，给出审核报告；
应用场景
  • 金融机构、券商、基金公司
  • 当日收盘后，对智能投顾交易监控并生成合规报告
投资策略持续合规实时审核系统

人工智能投顾策略，如果运用不当，易引起股市的趋同效应。我们系统即对智能投顾交易进行持续、实时监管，及时给出预警或控制，避免对股票市场造成风险。系统通过数据采集接口，获取智能投资策略的建仓、调仓、平仓等信息；通过第三方行情系统，获取行情交易数据。系统通过分析自身机构的交易趋同度、自身机构交易与整个行情交易的资金占比等信息，判断是否对股票市场造成冲击，并别给出无影响、警示或控制等建议。

功能列表
  • • 接收交易数据 通过数据接口获取智能投顾交易数据。
  • • 实时分析交易数据 实时分析交易数据趋同度，对市场影响等指标；
  • • 审核意见 以图形界面或接口的形式，给出审核意见；或对交易指令进行拦截以避免违规；
应用场景
  • 金融机构、券商、基金公司
  • 对智能投顾进行持续的、实时的交易监控
